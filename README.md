# ZezoKids Parental Control App

## Introduction
ZezoKids is a comprehensive parental control application designed to help parents monitor and manage their children's digital activities. Built with Flutter, this app ensures cross-platform compatibility and a seamless user experience.

## Features
- **Activity Monitoring**: Keep track of your child's app usage and online behavior.
- **Time Management**: Set limits on screen time and ensure healthy usage habits.
- **Web Filtering**: Block inappropriate websites and content to protect your child online.
- **Location Tracking**: Ensure your child's safety by tracking their location in real-time.
- **Behavioral Reports**: Receive weekly reports detailing your child's activity and app usage.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ahmedxtracom-boop/ZezoKids.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ZezoKids
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## Usage
### Setting Up Parental Controls
- Open the app and create a parent account.
- Add your child’s profile and customize the settings.
- Explore various options such as app limits, screen time, and content filters.

### Monitoring and Reporting
- Navigate to the monitoring tab to view real-time activity.
- Access reports to get insights into app usage and online behavior.

## Technologies Used
- **Flutter**: For building a beautiful user interface.
- **Firebase**: For authentication and real-time database functionality.
- **Geolocation Services**: To provide accurate location tracking.
